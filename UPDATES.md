# 🎉 功能更新說明 v2.0

## ✨ 新增功能總覽

根據你的需求，系統已全面升級！以下是所有新增的功能：

---

## 1️⃣ 自訂活動中心管理

### ✅ 現在可以做什麼：
- **在 App 內直接新增中心**（不用去 Supabase 後台了！）
- 編輯中心資訊
- 刪除不需要的中心
- 你的六個中心已經預設好了：
  - 文山樂齡中心
  - 聯新平鎮
  - 聯新中壢
  - 聯新復旦
  - 三峽中埔
  - 佳醫中壢

### 📍 如何使用：
```
1. 點擊底部「⚙️ 設定」
2. 選擇「活動中心」分頁
3. 輸入中心名稱
4. 點擊「+ 新增中心」
```

---

## 2️⃣ 自訂篩選條件

### ✅ 現在可以做什麼：
- **新增自己的季節選項**（例如：初春、盛夏）
- **新增自己的節日**（例如：耶誕節、母親節）
- **新增材料類型**（例如：串珠、毛線、木材）
- 篩選條件會自動同步到所有頁面

### 📍 如何使用：
```
1. 點擊底部「⚙️ 設定」
2. 選擇「篩選條件」分頁
3. 選擇類別（季節/節日/材料）
4. 輸入選項名稱
5. 點擊「+ 新增選項」
```

---

## 3️⃣ 刪除功能

### ✅ 可以刪除的內容：
- **作品**：滑鼠移到作品卡片上，左上角會出現 🗑️ 按鈕
- **教學記錄**：在作品詳情頁可以刪除特定記錄
- **活動中心**：在設定頁面刪除
- **長輩資料**：在設定頁面刪除

### ⚠️ 重要提醒：
- 刪除作品會同時刪除相關的教學記錄
- 刪除中心會刪除該中心的所有記錄
- 刪除前都會跳出確認對話框

---

## 4️⃣ 深色模式 Dark Mode 🌙

### ✅ 功能特色：
- 一鍵切換淺色/深色主題
- 自動記住你的偏好
- 支援系統主題跟隨
- 所有頁面統一切換

### 📍 如何切換：
```
1. 點擊底部「⚙️ 設定」
2. 在頁面最上方看到「深色模式」開關
3. 點擊切換
```

或者，系統會自動偵測你的裝置設定（如果手機開了深色模式，App 也會自動變深色）

## 📖 使用流程範例

### 情境 1：第一次使用
```
1. 打開 App → 點擊「⚙️ 設定」
2. 確認六個中心都已經建立好了
3. 點擊「長輩管理」分頁
4. 新增每個中心的長輩資料
5. 回到作品庫開始上傳作品
```

### 情境 2：自訂篩選條件
```
1. 點擊「⚙️ 設定」→「篩選條件」
2. 選擇「節日」類別
3. 輸入「母親節」
4. 點擊新增
5. 回到作品庫，篩選器會自動出現新選項
```

### 情境 3：刪除不要的作品
```
1. 在作品庫頁面
2. 滑鼠移到要刪除的作品上
3. 點擊左上角的 🗑️ 按鈕
4. 確認刪除
5. 完成！
```

### 情境 4：切換深色模式
```
1. 點擊「⚙️ 設定」
2. 看到最上方的「深色模式」開關
3. 點擊切換（會看到月亮 🌙 圖示）
4. 整個 App 立刻變成深色主題
5. 下次打開會記住你的選擇
```

## ❓ 常見問題

**Q: 舊的作品資料會不會不見？**
A: 不會！只是新增功能，舊資料完全保留。

**Q: 深色模式會影響照片嗎？**
A: 不會，照片顏色不變，只是背景和文字變深色。

**Q: 刪除中心會刪除作品嗎？**
A: 不會，只會刪除該中心的教學記錄，作品本身保留。

**Q: 可以恢復刪除的資料嗎？**
A: 無法恢復，刪除前請三思！建議定期備份 Supabase。
